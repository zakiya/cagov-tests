檢測日期是做檢測的日期。呈報日期是指病例被呈報給加州公共衛生局的日期。報告的日期數據不適用於2020年5月5日之前的檢測。檢測日期數據不包括州外和未知轄區檢測，總數可能與報告日期的數量不匹配。檢測呈陽性率的計算方法是陽性分子檢測的數量除以完成的總分子檢測數量。檢測呈陽性率包括州立和聯邦監獄、美國移民及海關執法機構、美國法警署拘留所及州立醫院部門機構中的檢測呈陽性率。之前呈報的是14日平均檢測呈陽性率。這些圖表所使用的人口分母來自加州財政部對2020年的人口預測。
