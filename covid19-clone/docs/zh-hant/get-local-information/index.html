熱線號碼

全州新型冠狀病毒熱線

833‑422‑4255 (833-4CA-4ALL)

州呼叫中心的服務時間為每週7天：

  • 週一至週五，太平洋時間上午8:00至下午8:00   
  • 週六和週日，太平洋時間上午8:00至下午5:00

此COVID-19新型冠狀病毒資訊熱線是我們不斷提供可靠、可信賴資訊和支援的工作的一部分。
