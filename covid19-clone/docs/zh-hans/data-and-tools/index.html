数据模型

加州COVID-19评估工具 (California COVID-19 Assessment Tool, CalCat) 是为州和地方进行应对工作提供信息的模型。您可以找到实时播报、预测和设想情况。

https://calcat.covid19.ca.gov

数据面板

全州病例统计和人口统计 

其中包括呈阳性病例、死亡病例和检测结果。

查看全州一览表

在开放数据门户网站 (Open Data Portal) 查看确诊病例数据及死亡病例数据

在开放数据门户网站查看检测数据

在开放数据门户网站查看可能存在的病例

在开放数据门户网站查看感染后数据

全州疫苗接种统计和人口统计 

其中包括对各群体已接种的COVID-19疫苗剂量数和疫苗接种进度的概述。

在开放数据门户网站查看疫苗接种进度数据

在开放数据门户网站按邮政编码查看疫苗接种进度数据

对加州内医院的影响 

其中包括各县疑似感染病毒及检测呈阳性的住院患者，包括加护病房 (Intensive Care Unit, ICU) 中的患者。

查看医院一览表

在开放数据门户网站查看医院数据

COVID-19疫情爆发案例数 

此数据集包含自2021年1月1日开始呈报给加州公共卫生局 (California Department of Public Health, CDPH)，并按不同情况分类的全州范围内的COVID-19疫情爆发案例数和相关病例数。

在开放数据门户网站查看COVID-19数据

个人防护装备 (Personal Protection Equipment, PPE) 分配

个人防护装备 (PPE) 由加州州长的后勤工作组进行分配。PPE用具包括N-95口罩、外科口罩、罩衣、护面罩及手套。

查看PPE分配一览表

对无家可归者的影响

Roomkey计划的影响和其他帮助无家可归的加州民众的措施。其中包括获取房间、房间入住和拖车式活动房屋分配。

查看无家可归者一览表
